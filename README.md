# TechFix

## Deskripsi:
TechFix adalah web untuk mencari lokasi tempat servis elektronik terdekat dari lokasi pengguna.

## Teknologi yang digunakan:
* php
* bootstrap
* mysql
* Google Maps API

## Fitur:
* Pilih Titik Koordinat
* Lokasi Tempat Servis Terdekat
* Informasi Tempat Servis
* Profil
* Maps
* Data Tempat Servis
